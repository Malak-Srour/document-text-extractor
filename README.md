# 📄 Document Text Extractor

A simple web application built with Gradio to extract text from PDF and Word documents.

## 🚀 Features

* Extract text from PDF files
* Extract text from Word (.docx) files
* Handle multiple file uploads
* Clean and structured output display

## 🛠️ Technologies Used

* Python
* Gradio
* pdfplumber
* python-docx

## 📦 Installation

```bash
pip install -r requirements.txt
```

## ▶️ Run the App

```bash
python app.py
```

## 📂 Supported Formats

* PDF (.pdf)
* Word (.docx)

## ⚠️ Notes

* `.doc` files are not fully supported (convert to `.docx`)
